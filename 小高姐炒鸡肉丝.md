# 切鸡肉
去掉鸡里脊 和筋膜
顺着肌肉的纹路切丝 或 切片

可以用冷冻的鸡胸肉 微波炉解冻 1.5min 更好切

# 腌鸡胸肉
加入
* 盐 1/8tsp 胡椒粉 1/8tsp
* 生抽 2tsp 
* 水 1tbsp 或 蛋清 1tbsp

稍微搅拌 不要太用力搅
加入 玉米淀粉 1/2 tsp

加入油 一会下锅更容易分开

# 炒鸡胸肉
炒到表面变色还没熟透的时候 加入1/4tsp 醋
然后直接关火出锅
用余温和第二次回锅的温度炒熟

# 调汁
* 盐 1/8tsp 老抽 1/8tsp
* 玉米淀粉 1/2tsp
* 鸡精或味精 一点
* 水 1/4cup

# 炒
1. 葱姜蒜 辣椒
2. 酱料: 豆豉 15粒左右 目测
3. 蔬菜: 可选青椒:一个 蘑菇:6个 目测 切成和肉同样大小
4. 炒好的肉
5. 调好的料汁

按顺序加入 料汁可以只放一部分
